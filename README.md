# Análisis y Visualización de Datos Trabajos 1 y 2(AVD)

Trabajo de la asignatura **Análisis y Visualización de Datos** que implementa diversos sistemas de búsqueda y análisis utilizando técnicas de Machine Learning y Deep Learning además de análisis exploratorio de datos.

## 📚 Notebooks del Proyecto

### **Bloque 1: Análisis Exploratorio de Datos**

- [1-1-spotify.ipynb](1-1-spotify.ipynb) - Análisis del dataset de Spotify
- [1-2-tuberculosis.ipynb](1-2-tuberculosis.ipynb) - Análisis exploratorio de tuberculosis

### **Bloque 2: Búsqueda y Deep Learning con Imágenes**

- [2-1-buscador-imagenes.ipynb](2-1-buscador-imagenes.ipynb) - Motor de búsqueda de imágenes similares
- [2-2-fen.ipynb](2-2-fen.ipynb) - Reconocimiento de posiciones de ajedrez (FEN)
- [2-3-1-imagen-deeplearning.ipynb](2-3-1-imagen-deeplearning.ipynb) - Buscador basado en Deep Learning (Parte 1)
- [2-3-2-imagen-deeplearning.ipynb](2-3-2-imagen-deeplearning.ipynb) - Buscador basado en Deep Learning (Parte 2)

### **Bloque 3: Búsqueda y Deep Learning con Audio**

- [3-1-buscador-audio.ipynb](3-1-buscador-audio.ipynb) - Motor de búsqueda de audios similares
- [3-2-audio-deeplearning.ipynb](3-2-audio-deeplearning.ipynb) - Motor de búsqueda de audios con Deep Learning multimodal

### **Bloque 4: Procesamiento de Texto**

- [4-1-analisis-sentimientos.ipynb](4-1-analisis-sentimientos.ipynb) - Análisis de sentimientos en reseñas
- [4-2-buscador-texto.ipynb](4-2-buscador-texto.ipynb) - Motor de búsqueda de documentos textuales con BERT

## 🔧 Dependencias

Consulte `requirements.txt` para la lista completa de dependencias. Las principales incluyen:
- PyTorch
- Scikit-learn
- Pandas & NumPy
- Librosa (procesamiento de audio)
- Matplotlib & Plotly (visualización)
